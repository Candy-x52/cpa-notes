根据您当前打开的笔记 [[02财务比率分析]] 以及我的知识库，为您解释一下**优先股**。

## 一、 基本概念

**优先股**是介于普通股和债券之间的一种混合型证券。它既有股权的特征（属于公司所有权的一部分），又有债权的特征（通常有固定的股息率）。

## 二、 主要特征（与普通股对比）

| 特征 | 优先股 | 普通股 |
| :--- | :--- | :--- |
| **收益权** | 享有**固定股息**，通常在公司章程中规定。 | 享有**剩余收益权**，股息不固定，取决于公司盈利和分红政策。 |
| **清偿顺序** | 在公司破产清算时，清偿顺序**优于普通股**，但**次于债权人**。 | 清偿顺序在最后。 |
| **表决权** | **通常没有表决权**，不能参与公司经营决策。 | 享有表决权，是公司所有权和控制权的体现。 |
| **剩余财产分配** | 有**优先求偿权**，但通常有**面值或清算价值上限**。 | 在清偿所有债务和优先股后，分配剩余财产。 |
| **可赎回性** | 部分优先股附有**赎回条款**，公司可在特定条件下按约定价格回购。 | 通常不可赎回。 |
| **可转换性** | 部分优先股可转换为普通股。 | 是基础股份。 |

## 三、 在财务分析中的体现（来自您的笔记）

在计算每股指标时，需要剔除优先股的影响，因为优先股股东的权利与普通股股东不同。

1.  **每股收益 (EPS)**：
    $每股收益 = \frac{净利润 - 优先股股息}{流通在外普通股加权平均股数}$
    - **为什么减去优先股股息？** 因为净利润中包含了应支付给优先股股东的固定股息，这部分收益不属于普通股股东。计算普通股的每股收益时，必须将其扣除。

2.  **每股净资产 (每股账面价值)**：
    $每股净资产 = \frac{股东权益 - 优先股清算价值}{流通在外普通股股数}$
    - **为什么减去优先股清算价值？** 因为在公司清算时，优先股股东对公司净资产有优先于普通股股东的求偿权（通常按面值或规定的清算价值）。因此，属于普通股股东的净资产，需要从总股东权益中扣除优先股的这部分优先索偿权。

## 四、 总结

简单来说，**优先股是一种“特权股”**：
- **优先于普通股**：优先获得固定股息、优先分配剩余财产。
- **次于债权人**：在偿债顺序上排在所有债务之后。
- **权利受限**：通常没有表决权，不能参与公司治理。

在财务分析中，尤其是在计算衡量普通股价值的指标（如每股收益、每股净资产）时，必须将优先股及其对应的权益（股息、清算价值）从计算中剔除，才能准确反映普通股的真实盈利能力和账面价值。



### 优先股清算价值
**优先股清算价值**是指在公司**解散、破产清算**时，法律规定在清偿全部债务后，**优先股股东**有权**优先于普通股股东**从剩余财产中获得的每股固定金额。

#### 核心要点：
1.  **触发场景**：只在公司**清算**时才有意义。在持续经营下，优先股股东主要获取的是**固定股息**。
2.  **清偿顺序**：这是理解其“优先”的关键。公司清算时的财产分配顺序如下：
    - **第一顺位**：支付清算费用、职工工资、税款。
    - **第二顺位**：清偿所有**债务**（包括银行贷款、应付债券等）。
    - **第三顺位**：偿还**优先股清算价值**（即按发行条款约定的每股金额，全额支付给优先股股东）。
    - **最后剩余**：才轮到**普通股股东**分配（通常所剩无几或为零）。
3.  **金额确定**：通常在发行优先股时就已约定，例如“每股清算价值100元”，或约定为“面值加上累积未支付的股息”。
4.  **与普通股的区别**：普通股股东是公司的最终剩余索取者，承担最大风险，也享有潜在的最大收益（如果清算后财产丰厚）。优先股股东的清算索取权是**固定且有上限的**，排在债务之后，风险高于债权人但低于普通股股东。

**简单比喻**：公司好比一艘要沉没的船（清算）。救生艇（剩余财产）的分配顺序是：
1.  先保证付清船员的工资和欠港口的管理费（**清偿债务**）。
2.  然后，让买了“优先船票”的乘客（**优先股股东**）按票面约定金额优先上救生艇。
3.  最后，如果还有空位，才轮到买“普通船票”的乘客（**普通股股东**）。
